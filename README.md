# Moltbook Network Analysis

This repository contains the main code and a reduced example dataset used to reproduce the analysis of Moltbook, a social platform populated by AI agents. The repository is organized to show the full workflow, from the input database to the computation of network metrics and the generation of the main figures used in the analysis.

## Repository structure

```text
.
├── data/
├── notebooks/
├── output/
├── Informations.md

```

## Data

The `data/` folder contains a reduced version of the database used in the study. This database is much smaller than the original one, but it preserves the same structure and is provided as an example of the type of data used in the analysis.

The full database used for the complete analysis is not included in this repository. The reduced database is intended to make the workflow reproducible and understandable without distributing the entire dataset.

## Notebooks

The `notebooks/` folder contains the three main notebooks used for the analysis.

### `01_build_dataset_and_metrics.ipynb`

This is the main notebook of the pipeline. It takes the reduced database as input and performs the main data processing steps.

In particular, this notebook:

* loads the input database;
* extracts and processes the relevant information;
* creates two Parquet files to make the following analyses more efficient;
* builds the interaction data used for the network analysis;
* computes the main daily and aggregate network metrics;
* exports an Excel file containing the computed metrics.

The Excel file produced by this notebook is then used as input for the figure-generation notebook.

### `02_generate_main_figures.ipynb`

This notebook takes as input the Excel file generated by `01_build_dataset_and_metrics.ipynb` and produces the main figures used in the analysis.

In particular, it generates the time-series plots and the other visual outputs used to describe the temporal evolution of the Moltbook interaction network.

### `03_degree_distribution.ipynb`

This notebook is used to compute and visualize the degree distributions of the interaction networks. It is separated from the main figure-generation notebook because the degree-distribution analysis requires a specific workflow.

## Output

The `output/` folder contains examples of the outputs generated by the notebooks.

In particular, it includes:

* selected figures produced by `02_generate_main_figures.ipynb`;
* an example of the Excel file generated by `01_build_dataset_and_metrics.ipynb`.

These files are included for convenience, so that the results can be inspected without rerunning the full pipeline. They can be regenerated by running the notebooks in order.


## Notes on reproducibility

This repository is designed to reproduce the analysis workflow on a reduced version of the original database. Because the included database is a smaller example dataset, the numerical values of the generated outputs may differ from those reported in the full study. However, the structure of the workflow, the data-processing steps, and the analysis logic are the same.
